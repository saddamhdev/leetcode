```java
public int binarySearch(int[] nums, int target) {
    int left = 0;
    int right = nums.length - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (nums[mid] == target) {
            return mid; // index found
        } 
        else if (target < nums[mid]) {
            right = mid - 1; // go left
        } 
        else {
            left = mid + 1; // go right
        }
    }

    return -1; // not found
}
```
