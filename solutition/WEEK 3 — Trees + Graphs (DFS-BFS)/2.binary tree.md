## **B. Breadth First Search (BFS)**

Also called **Level Order Traversal**
Uses a **Queue**.

```
levelOrder(root):
    if root == null: return
    queue = [root]
    while queue:
        node = queue.pop(0)
        visit(node)
        if node.left: queue.append(node.left)
        if node.right: queue.append(node.right)
```
