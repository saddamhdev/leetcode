# Java-তে লিংকড লিস্ট সমস্যা সমাধান

## মৌলিক নোড ক্লাস:

```java
class Node {
    int data;
    Node next;
    
    Node(int data) {
        this.data = data;
        this.next = null;
    }
}
```

## সাধারণ অপারেশন:

**১. ট্রাভার্স (সব নোড দেখা):**
```java
void traverse(Node head) {
    Node current = head;
    while(current != null) {
        System.out.print(current.data + " -> ");
        current = current.next;
    }
    System.out.println("null");
}
```

**২. খুঁজে পাওয়া (Search):**
```java
boolean search(Node head, int key) {
    Node current = head;
    while(current != null) {
        if(current.data == key) return true;
        current = current.next;
    }
    return false;
}
```

**৩. যোগ করা (শেষে - Append):**
```java
void append(Node head, int data) {
    Node newNode = new Node(data);
    Node current = head;
    
    while(current.next != null) {
        current = current.next;
    }
    current.next = newNode;
}
```

**৪. মুছে ফেলা (Delete):**
```java
Node delete(Node head, int key) {
    // যদি প্রথম নোড মুছতে হয়
    if(head.data == key) {
        return head.next;
    }
    
    Node current = head;
    while(current.next != null) {
        if(current.next.data == key) {
            current.next = current.next.next;
            return head;
        }
        current = current.next;
    }
    return head;
}
```

**৫. রিভার্স করা:**
```java
Node reverse(Node head) {
    Node prev = null;
    Node current = head;
    
    while(current != null) {
        Node next = current.next;  // পরবর্তী সংরক্ষণ করুন
        current.next = prev;        // দিক বদল করুন
        prev = current;             // এগিয়ে যান
        current = next;
    }
    return prev;
}
```

**৬. মধ্যবিন্দু খুঁজা:**
```java
Node findMiddle(Node head) {
    Node slow = head;
    Node fast = head;
    
    while(fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;  // মধ্য নোড
}
```

**৭. লুপ চেক করা:**
```java
boolean hasCycle(Node head) {
    Node slow = head;
    Node fast = head;
    
    while(fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if(slow == fast) return true;  // লুপ আছে
    }
    return false;
}
```

## সম্পূর্ণ উদাহরণ:

```java
public class LinkedList {
    Node head;
    
    public static void main(String[] args) {
        LinkedList list = new LinkedList();
        
        // নোড যোগ করা
        list.head = new Node(10);
        list.head.next = new Node(20);
        list.head.next.next = new Node(30);
        
        // ট্রাভার্স
        traverse(list.head);  // 10 -> 20 -> 30 -> null
        
        // খুঁজা
        System.out.println(search(list.head, 20));  // true
        
        // মুছা
        list.head = delete(list.head, 20);
        traverse(list.head);  // 10 -> 30 -> null
    }
}
```
