# 1️⃣ **Opposite Direction Two Pointers**

ব্যবহার: যতক্ষণ দুই দিক থেকে তুলনা করতে হবে

### উদাহরণ সমস্যা:

* Reverse String
* Valid Palindrome
* Container With Most Water
* Two Sum II (sorted)
* Remove duplicates (variant)

### Java টেমপ্লেট:

```java
int left = 0;
int right = arr.length - 1;

while (left < right) {
    // কাজ
    left++;
    right--;
}
```
